#  Interview Craft Studio

An AI-powered platform that helps job seekers prepare smarter for interviews by generating personalized interview questions, identifying skill gaps, and providing a structured roadmap.

---

## 🌟 Features

* 📄 Upload Resume (PDF)
* 🧾 Add Job Description
* 🧑 Provide Self Description
* 🤖 AI-powered analysis using Gemini API
* 🎯 Generates:

  * Technical Interview Questions
  * Behavioral Interview Questions
  * Skill Gap Analysis
  * Personalized Preparation Roadmap

---

## 🧠 How It Works

1. User uploads resume (PDF)
2. Enters job description
3. Adds self-description
4. Backend processes resume content
5. Gemini AI analyzes all inputs
6. System generates:

   * Interview questions
   * Skill gaps
   * Roadmap for preparation

---

### Backend

* Gemini API (Google AI)

## ⚙️ Installation & Setup

### 1. Clone the repository

```bash
git clone https://github.com/your-username/interview-craft-studio.git
cd interview-craft-studio
```

### 2. Install dependencies

#### Backend

```bash
cd backend
npm install
```

#### Frontend

```bash
cd frontend
npm install
```

---

## 🔑 Environment Variables

Create a `.env` file in backend:

```
GEMINI_API_KEY=your_api_key
PORT=3000
```

> Note: On Render, PORT is automatically assigned.

---

## ▶️ Run Locally

### Backend

```bash
npm run dev
```

### Frontend

```bash
npm start
```
